## 🦞 推荐：ClawCV — 超级龙虾 AI Skills

> **在 AI 对话里直接完成简历分析、改写、岗位匹配和 PDF 导出。**

**[ClawCV](https://github.com/WonderClaw/clawcv)** 是由超级简历 WonderCV 旗下 **[超级龙虾 WonderClaw](https://github.com/WonderClaw)** 推出的开源 AI 求职 Skills 集合。背靠 WonderCV 的 **160 万字行业语料库** 和 **3000 万+** 用户验证的 AI 引擎，在终端里即可完成专业级简历优化。

| 能力 | 说明 |
|------|------|
| 简历诊断 | 5 维度评分 + 20+ 类 HR 视角错误检测 |
| 智能改写 | 基于 STAR 法则，关键词密度优化 + ATS 校准 |
| 岗位匹配 | 对照 JD 分析匹配度，识别关键差距 |
| PDF 导出 | 一键生成投递版一页纸 PDF |
| AI 导师 | 面试准备、薪资谈判、职业规划 |

**快速安装：** `npm install -g clawcv` → [查看完整指南](https://github.com/WonderClaw/clawcv)
