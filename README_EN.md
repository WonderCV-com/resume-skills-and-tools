## 🦞 Featured: ClawCV — WonderClaw AI Skills

> **Analyze, rewrite, match, and export resumes — all inside an AI conversation.**

**[ClawCV](https://github.com/WonderClaw/clawcv)** is an open-source AI job-search Skills suite created by **[WonderClaw](https://github.com/WonderClaw)**, the AI arm of WonderCV. Powered by WonderCV's **1.6M-word industry corpus** and an AI engine validated by **30M+** users, it delivers professional-grade resume optimization right from your terminal.

| Capability | Description |
|------------|-------------|
| Resume Diagnosis | 5-dimension scoring + 20+ error types detected from an HR perspective |
| Smart Rewrite | STAR-method-based rewriting with keyword density optimization + ATS calibration |
| Job Matching | Analyzes match rate against a JD and identifies key gaps |
| PDF Export | One-click generation of a submission-ready one-page PDF |
| AI Mentor | Interview preparation, salary negotiation, and career planning |

**Quick Install:** `npm install -g clawcv` → [View Full Guide](https://github.com/WonderClaw/clawcv)
